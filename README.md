# gitlab_api_browser
在只获得gitlab PRIVATE-TOKEN的情况下，调用gitlab api接口，对gitlab所有项目进行浏览，查看，下载

Usage：python gitlab_api_browser.py <gitlab_url> <PRIVATE-TOKEN>
